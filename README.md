## 🚀 Quick Launch (15 Minutes)

### 1. Deploy to Vercel (5 min)

```bash
# Clone this repo
git clone https://github.com/YOUR_USERNAME/calcninja.git
cd calcninja

# Or if you started from scratch:
git init
git add .
git commit -m "CalcNinja launch"
git branch -M main
git push -u origin main
```

Then go to **https://vercel.com** → Import project from GitHub → Deploy ✅

**Your site lives at:** `https://calcninja.vercel.app`

### 2. Verify on Google Search Console (5 min)

1. https://search.google.com/search-console
2. Add property → URL Prefix
3. Enter: `https://calcninja.vercel.app`
4. Verify using HTML file upload
5. Submit sitemap: `https://calcninja.vercel.app/sitemap.xml`

### 3. Apply for Google AdSense (10 min)

1. https://adsense.google.com
2. Add your site
3. Wait 1-3 days for approval
4. Add code to `<head>` of index.html
5. Start earning! 💰

**See [QUICK_START.md](QUICK_START.md) for detailed step-by-step guide with screenshots.**
